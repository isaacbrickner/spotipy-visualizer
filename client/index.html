Feature Analysis

This section is an analysis and visualization of the features on your top 20 Spotify tracks.

  • acousticness: A confidence measure from 0.0 to 1.0 of whether the track is acoustic. 1.0 represents high confidence the track is acoustic. Range: 0 - 1. Example: 0.00242

  • analysis_url: A URL to access the full audio analysis of this track. An access token is required to access this data.
    Example: "https://api.spotify.com/v1/audio-analysis/2takcwOaAZWiXQijPHIx7B"

  • danceability: Danceability describes how suitable a track is for dancing based on a combination of musical elements including tempo, rhythm stability, beat strength, and overall regularity. A value of 0.0 is least danceable and 1.0 is most danceable.
    Example: 0.585

  • duration_ms: The duration of the track in milliseconds.
    Example: 237040

  • energy: Energy is a measure from 0.0 to 1.0 and represents a perceptual measure of intensity and activity. Typically, energetic tracks feel fast, loud, and noisy.
    Example: 0.842

  • id: The Spotify ID for the track.
    Example: "2takcwOaAZWiXQijPHIx7B"

  • instrumentalness: Predicts whether a track contains no vocals. "Ooh" and "aah" sounds are treated as instrumental in this context.
    Example: 0.00686

  • key: The key the track is in. Integers map to pitches using standard Pitch Class notation.
    Example: 9

  • liveness: Detects the presence of an audience in the recording.
    Example: 0.0866

  • loudness: The overall loudness of a track in decibels (dB).
    Example: -5.883

  • mode: Mode indicates the modality (major or minor) of a track.
    Example: 0

  • speechiness: Speechiness detects the presence of spoken words in a track.
    Example: 0.0556

  • tempo: The overall estimated tempo of a track in beats per minute (BPM).
    Example: 118.211

  • time_signature: An estimated time signature.
    Example: 4

  • track_href: A link to the Web API endpoint providing full details of the track.
    Example: "https://api.spotify.com/v1/tracks/2takcwOaAZWiXQijPHIx7B"

  • valence: A measure from 0.0 to 1.0 describing the musical positiveness conveyed by a track. Range: 0 - 1.
    Example: 0.428
